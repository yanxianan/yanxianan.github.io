函数指针和指针函数

发表于 2026-05-20 分类于 C_C++
本文字数： 1.1k 阅读时长 ≈ 2 分钟
本文详细介绍函数指针和指针函数。

函数指针和指针函数是 C/C++ 中名称相似但含义完全不同的两个概念。简单说：

  • 指针函数：本质是函数，返回值是一个指针。
  • 函数指针：本质是指针，指向的是函数。

下面分别说明它们的写法和用法。

3. 核心区别对比

项目 指针函数 函数指针
本质 函数 指针变量
返回值 返回指针类型的数据 变量本身存储函数地址，无“返回值”概念
声明关键 int* f(int a) int (*p)(int a)
应用场景 返回数组、动态内存、字符串等 回调函数、策略模式、运行时选择函数
结合性 * 与类型结合，表明返回指针 * 与变量名结合，表明是指针变量

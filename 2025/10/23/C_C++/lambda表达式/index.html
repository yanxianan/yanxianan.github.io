本文详细介绍c++的lambda表达式。

C++ 的 lambda 表达式（Lambda Expression）是 C++11 引入的一项重要特性，用于定义匿名函数对象（也称为闭包）。它提供了一种简洁、内联的方式来编写函数，特别适用于 STL 算法、回调函数、并行编程等场景。

四、lambda 的类型与存储

每个 lambda 表达式都会生成一个唯一的匿名类类型（闭包类型），该类型重载了 operator()。

因此，不能直接用 decltype(f) 声明另一个 lambda（除非是同一表达式），但可以用：

  • std::function<R(Args...)> 存储（有运行时开销）
  • 模板参数自动推导（无开销）

六、注意事项与陷阱

1. 悬空引用（Dangling Reference）

应使用 [=] 或 [x] 按值捕获。

2. mutable 关键字

3. 性能考虑

  • 按值捕获大对象可能有拷贝开销（可考虑移动捕获，C++14 起支持初始化捕获）
  • std::function 有类型擦除开销，优先使用模板或直接传递 lambda

七、总结

特性 说明
匿名函数 无需命名，内联定义
闭包 可捕获外部变量
类型 每个 lambda 有唯一匿名类型
用途 STL、回调、线程、简化代码
安全 注意引用生命周期、mutable、移动语义

Lambda 表达式极大提升了 C++ 的表达能力和编程效率，是现代 C++ 编程不可或缺的工具。掌握其捕获机制和使用场景，能写出更简洁、高效、安全的代码。

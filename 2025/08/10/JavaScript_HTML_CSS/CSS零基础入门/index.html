CSS 零基础入门：从样式定义到页面美化的完整指南

发表于 2025-08-10 更新于 2025-10-23 分类于 JavaScript_HTML_CSS
本文为零基础学习者提供 CSS 完整入门教程，涵盖 CSS 的基本概念、引入方式、选择器、常用样式属性及实战案例，帮助读者快速掌握网页美化技能。

CSS（Cascading Style Sheets，层叠样式表）是用于控制 HTML 内容外观和布局的核心 Web 技术。它将网页的结构（HTML）与表现（样式）分离，使开发者能高效地设计美观、响应式、可维护的网站。

一、CSS 的基本概念

1. 什么是 CSS？

  • CSS 是一种样式语言，不是编程语言。
  • 它通过选择器（Selector）选中 HTML 元素，并为其设置样式规则（如颜色、字体、间距、布局等）。

2. “层叠”（Cascading）的含义

  • 多个 CSS 规则可能作用于同一个元素。
  • 浏览器根据优先级规则（来源、选择器特异性、!important 等）决定最终应用哪条样式。
  • 样式可以继承（如字体颜色从父元素传给子元素）。

二、CSS 的引入方式

1. 行内样式（不推荐）

  • 优先级最高，但难以维护，违反“结构与样式分离”原则。

3. 外部样式表（推荐）

四、选择器（Selectors）

选择器用于“选中”要样式化的 HTML 元素。

1. 基础选择器

类型 示例 说明
元素选择器 p 选中所有 <p> 元素
类选择器 .highlight 选中 class="highlight" 的元素
ID 选择器 #header 选中 id="header" 的元素（唯一）
通配符 * 选中所有元素

2. 组合与关系选择器

示例 说明
div p 后代选择器：div 内所有 p（无论嵌套多深）
div > p 子选择器：div 的直接子元素 p
h1 + p 相邻兄弟：紧跟在 h1 后的 p
h1 ~ p 通用兄弟：h1 之后所有同级 p

3. 伪类与伪元素

类型 示例 说明
伪类 a:hover 鼠标悬停时的链接
伪类 p:first-child 作为第一个子元素的 p
伪元素 p::first-line 段落第一行
伪元素 ::before, ::after 在元素前后插入内容

注意：伪类用单冒号（:hover），伪元素用双冒号（::before），但为兼容旧浏览器，单冒号也常被接受。

4. 布局相关

  • display: 控制元素显示类型
    • block（独占一行，如 div）
    • inline（行内，如 span）
    • inline-block（行内但可设宽高）
    • flex（弹性布局）
    • grid（网格布局）
    • none（隐藏元素）
  • position: 定位方式
    • static（默认）
    • relative（相对自身原位置偏移）
    • absolute（相对于最近的非 static 祖先定位）
    • fixed（相对于视口固定）
    • sticky（滚动时“粘”在视口某位置）
  • float（旧布局方式，现多用 Flex/Grid 替代）

六、现代布局技术

1. Flexbox（弹性盒子）

适合一维布局（行或列）。

2. CSS Grid（网格布局）

适合二维布局（行 + 列）。

✅ 推荐：简单布局用 Flex，复杂网格用 Grid。

九、层叠与优先级（Cascading & Specificity）

当多个规则冲突时，浏览器按以下顺序决定：

  1. 来源顺序：后写的覆盖先写的（同优先级下）

  2. 选择器特异性

    （Specificity）：

    • !important > 行内样式 > ID > 类/属性/伪类 > 元素/伪元素
    • 特异性计算：[内联, ID, 类, 元素]，如 #nav .item a → [0,1,1,1]

  3. 继承：某些属性（如 color, font）会从父元素继承

❗ 尽量避免 !important，它会破坏层叠逻辑。

十、最佳实践

  1. 语义化类名：.btn-primary 而非 .red-button
  2. 使用外部 CSS 文件
  3. 全局重置或标准化样式（如 normalize.css）
  4. 采用 BEM 命名法（Block__Element–Modifier）避免命名冲突
  5. 响应式优先：移动优先 + 媒体查询
  6. 使用 Flex/Grid 替代 float 和 table 布局
  7. 压缩 CSS 文件（生产环境）
  8. 合理使用 CSS 变量管理主题

十一、工具与扩展

  • 预处理器：Sass、Less（支持变量、嵌套、函数等）
  • PostCSS：自动加浏览器前缀（如 autoprefixer）
  • CSS 框架：Bootstrap、Tailwind CSS（加速开发）
  • 开发者工具：浏览器 F12 可实时调试 CSS

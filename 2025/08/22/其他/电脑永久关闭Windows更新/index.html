如何通过修改注册表关闭Windows更新（Windows 10/11）

一、操作步骤

步骤1：打开注册表编辑器

  1. 按下 Win + R 键，打开“运行”窗口。
  2. 输入 regedit，然后按 “确定”。
  3. ·如果弹出用户账户控制（UAC）提示，点击“是”继续。

步骤3：创建或修改注册表值

步骤4：禁用Windows Update服务（可选）

  1. 按 Win + R，输入 services.msc，回车。
  2. 找到 Windows Update 服务。
  3. 右键 → “属性” → 将“启动类型”改为 禁用。
  4. 如果服务正在运行，先点击“停止”，再点击“应用”。

⚠️ 完全禁用服务可能导致系统长期不打补丁，存在安全风险。

步骤5：重启电脑

重启后，设置生效。系统将不再自动下载和安装更新。

三、恢复Windows更新的方法

如果你想重新启用更新：

  1. 删除 HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate 整个项。
  2. 打开服务管理器，将 Windows Update 服务的启动类型改为 自动，并启动服务。
  3. 重启电脑。

四、替代方案（推荐）

与其完全关闭更新，不如使用更安全的替代方法：

  • 使用 组策略编辑器（仅限Windows专业版及以上）：
    路径：计算机配置 → 管理模板 → Windows组件 → Windows更新
  • 使用 Windows更新暂停功能（设置 → 更新与安全 → 暂停更新7天，可多次延长）
  • 配置 计量连接，限制后台更新流量

五、注意事项

  • 关闭更新会使系统暴露于安全漏洞中，建议定期手动检查更新。
  • 某些企业环境或学校网络可能强制启用更新策略。
  • 修改注册表前请确保了解每项设置的含义。

总结：通过注册表可以彻底关闭Windows更新，但需权衡便利性与系统安全。建议采用“通知更新”模式（AUOptions=2），在保持安全的同时避免打扰。

本文适用于Windows 10 和 Windows 11 系统。操作有风险，谨慎执行。

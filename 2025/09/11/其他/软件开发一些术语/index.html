一些术语

发表于 2025-09-11 更新于 2025-09-12 分类于 其他
本文为阅读论文遇见的一些术语

异构组合：

“异构组合”是一个在计算机、电子工程、系统架构等领域中常见的术语，它的核心含义是：

由不同类型（不同架构、不同技术、不同功能）的组件组合在一起，协同工作，形成一个完整的系统。

RFDC：

RFDC 是 Radio Frequency Data Converter 的缩写，中文意思是 射频数据转换器。

它是现代高性能通信系统、雷达、5G 基站、软件定义无线电（SDR）、测试测量设备等领域中的核心芯片技术，能够直接对射频（RF）信号进行高速模数（ADC）或数模（DAC）转换，跳过传统中频处理环节，实现更高效、更灵活的信号处理。

一、RFDC 是什么？

🔹 定义：

RFDC 是一种高带宽、高采样率的数据转换器（ADC 或 DAC），可以直接处理 GHz 级别的射频频率信号，无需像传统系统那样先将信号下变频到中频（IF）再进行数字化。

  • RFDC ADC：将模拟射频信号直接转换为数字信号（A/D 转换）
  • RFDC DAC：将数字信号直接生成射频信号（D/A 转换）

✅ 简单说：它是一个“能直接处理射频信号的高速 ADC/DAC”。

软件无线电理念：

“软件无线电”（Software Defined Radio，简称 SDR）的核心理念可以用一句话概括：

尽可能用软件来实现无线电系统的功能，而不是依赖固定的硬件电路。

软件无线电的五大核心理念

理念 说明
1. 硬件通用化 使用宽频带天线、宽带射频前端、高速 ADC/DAC，支持多种频率和带宽。
2. 功能软件化 所有信号处理（调制、解调、滤波、编码）由软件或可编程逻辑（FPGA）实现。
3. 可重构性 设备功能可通过软件更新动态改变，无需更换硬件。
4. 开放性与可编程性 提供 API 或开发环境，允许用户自定义通信协议。
5. 多模多频兼容 一台设备支持多种通信标准（如 GSM、LTE、Wi-Fi、LoRa）。

功能解耦：

“功能解耦”（Functional Decoupling）是一个在 软件工程、系统架构、硬件设计、通信系统 等领域广泛使用的概念，它的核心思想是：

将原本紧密关联、互相依赖的功能模块拆分开来，使它们可以独立开发、维护和升级，降低系统复杂性。

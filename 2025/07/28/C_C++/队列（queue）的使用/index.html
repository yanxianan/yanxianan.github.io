C++ 中的队列 queue 详解

发表于 2025-07-28 分类于 C_C++
本文详细介绍 C++ 中 STL 队列 queue 的概念、特性、常用成员函数及使用示例，帮助读者快速掌握 queue 的使用方法。

队列是一种常见的数据结构，遵循 “先进先出”（FIFO）原则。在 C++ 的 STL（标准模板库）中，queue 容器为我们提供了实现队列功能的便捷方式。本文将详细介绍 C++ 中 queue 的特性、常用操作及使用示例。

一、queue 的基本概念

queue 是 C++ STL 中的一个容器适配器，它基于其他容器（默认是 deque）实现，提供了队列的基本操作。其核心特点是元素的插入在队尾进行，元素的删除和访问在队头进行，严格遵循先进先出的规则。

三、queue 的常用成员函数

  1. 入队操作：push()
    • 功能：在队列的队尾插入一个元素。
    • 示例：q.push(10);
  1. 出队操作：pop()
    • 功能：删除队列的队头元素，无返回值。
    • 注意：使用前需判断队列是否为空，否则可能导致未定义行为。
    • 示例：if (!q.empty()) q.pop();
  1. 访问队头元素：front()
    • 功能：返回队头元素的引用。
    • 示例：int frontVal = q.front();
  1. 访问队尾元素：back()
    • 功能：返回队尾元素的引用。
    • 示例：int backVal = q.back();
  1. 判断队列是否为空：empty()
    • 功能：若队列为空，返回 true；否则返回 false。
    • 示例：if (q.empty()) cout << “队列为空” << endl;
  1. 获取队列元素个数：size()
    • 功能：返回队列中元素的数量。
    • 示例：cout << “队列元素个数：” << q.size() << endl;

五、queue 的注意事项

  1. queue 没有迭代器，无法直接遍历队列中的所有元素，只能通过不断出队的方式访问。

  2. 使用front()和pop()函数前，务必先用empty()判断队列是否为空，避免操作空队列导致错误。

  3. queue 的底层容器默认是 deque，也可以指定为 list，但不能是 vector，因为 vector 不支持高效的头删操作。

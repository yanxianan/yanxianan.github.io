MySQL 基础和安装配置

1. MySQL 基本概念

1.1 什么是 MySQL

MySQL 是一个开源的关系型数据库管理系统（RDBMS），它使用结构化查询语言（SQL）来管理数据。MySQL 因其可靠性、稳定性、简单易用等特点，广泛应用于 Web 应用程序、数据分析、内容管理系统等领域。

1.2 MySQL 的特点

  • 开源免费：源代码开放，可自由使用和修改
  • 跨平台：支持 Linux、Windows、Mac 等多个操作系统
  • 可靠稳定：支持事务、外键等特性，数据安全性高
  • 性能优秀：查询速度快，适合大数据量处理
  • 易于使用：安装配置简单，学习成本低
  • 支持多种存储引擎：如 InnoDB、MyISAM 等

1.3 数据库系统的核心概念

核心概念解释：

  • 数据库（Database）：相关表的集合，用于存储特定应用的所有数据
  • 表（Table）：具有相同结构的行的集合，是数据存储的基本单位
  • 行（Row）：表中的一条记录，代表一个完整的数据对象
  • 列（Column）：表中的一个字段，代表数据的一个属性

2. MySQL 版本和历史

2.1 主要版本演进

版本 发布时间 特点
5.0 - 5.7 2005-2013 稳定实用，被广泛使用
5.7 2013-2021 性能优化，支持 JSON
8.0 2018 至今 新特性丰富，性能提升

2.2 MySQL 8.0 的重要变化

  • 默认认证插件从 mysql_native_password 改为 caching_sha2_password
  • 支持窗口函数和公共表表达式（CTE）
  • JSON 功能增强
  • 隐藏索引
  • 性能优化

3. MySQL 安装配置

3.1 Windows 安装

方式一：MSI 安装程序

  1. 下载安装程序

    • 访问 MySQL 官网
    • 选择 Windows 版本下载

  2. 运行安装程序

  3. 选择安装类型

    • Developer Default（推荐新手）
    • Server only
    • Custom

  4. 配置 MySQL 服务

    • Port：默认 3306
    • MySQL Server 类型：Development Machine（开发）或 Server（服务器）
    • 认证方法：选择推荐的加密密码

  5. 创建 MySQL 用户账户

    • 用户名：默认 root
    • 设置强密码

  6. 完成安装

    • MySQL Server 作为 Windows 服务自动启动

  3. 创建 my.ini 配置文件

7. 常见问题排查

7.1 连接失败

问题：ERROR 2003: Can't connect to MySQL server

解决方案：

  • 检查 MySQL 服务是否启动
  • 检查主机名和端口是否正确
  • 检查防火墙是否阻止连接

7.2 密码错误

问题：ERROR 1045: Access denied for user 'root'@'localhost'

解决方案：

  • 确认输入的密码正确
  • 如果忘记 root 密码，需要以 –skip-grant-tables 模式启动 MySQL（高级话题）

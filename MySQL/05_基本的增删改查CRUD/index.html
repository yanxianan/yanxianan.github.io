5. 事务（Transaction）

5.1 事务基本概念

事务是一组 SQL 操作的集合，具有原子性，要么全部执行，要么全部不执行。

ACID 特性：

  • A (Atomicity)：原子性，全部或都不执行
  • C (Consistency)：一致性，从一个一致状态到另一个一致状态
  • I (Isolation)：隔离性，并发事务相互隔离
  • D (Durability)：持久性，提交后永久保存

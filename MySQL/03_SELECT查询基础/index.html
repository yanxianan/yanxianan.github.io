11. SELECT 最佳实践

  • 不要使用 SELECT *：明确指定需要的列，提高性能和可读性
  • 添加 WHERE 条件：避免查询全表数据
  • 合理使用 LIMIT：特别是在大表上进行分页
  • 使用有意义的别名：提高查询结果的可读性
  • 避免在 WHERE 中对列进行函数处理：会导致索引失效
  • 定期检查查询性能：使用 EXPLAIN 分析查询计划

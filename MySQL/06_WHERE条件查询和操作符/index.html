WHERE 条件查询和操作符

1. WHERE 子句的作用

WHERE 子句用于筛选满足条件的数据行，是 SQL 查询中最常用的条件语句。

执行流程：

  1. 读取表中的每一行
  2. 判断是否满足 WHERE 条件
  3. 满足条件的行返回到结果集中

2. 比较操作符

操作符 说明 示例
= 等于 WHERE age = 20
<> 或 != 不等于 WHERE age <> 20
> 大于 WHERE age > 20
< 小于 WHERE age < 20
>= 大于等于 WHERE age >= 20
<= 小于等于 WHERE age <= 20
